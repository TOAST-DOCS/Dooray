## TOAST 상품 가입하기

무료 체험 종료 후에도 계속 Dooray!를 이용하고 싶으시면, **TOAST 상품 가입**을 진행해 주세요.  
무료체험 종료 후 7일 이내(가입 37일 이내)까지 [TOAST 상품 가입]을 하지 않으시면 이후 가입은 자동 해지됩니다.   
Dooray! TOAST 상품은 가격에 따라 아래와 같이 분류됩니다. 

![](http://static.toastoven.net/prod_dooray_project/common/common14.png)
<center>[그림1]</center>


무료 상품을 이용하실 경우에도 기한 내에 **반드시 TOAST 상품 가입 신청**을 하셔야 계속 이용할 수 있습니다.

**1.[TOAST 상품 가입] 클릭**
조직관리 화면 상단 또는 무료 체험 관련 사전 안내 메일을 통해 [TOAST 상품 가입] 버튼을 클릭해 주세요.

**2.안내 확인 -> [TOAST 상품 가입] 클릭**

![](http://static.toastoven.net/prod_dooray_project/common/common10.png)
<center>[그림2]</center>


무료체험 중 사용한 서비스가 포함되지 않은 TOAST 상품을 선택하면 **미포함 서비스의 데이터는 모두 삭제** 됩니다. 
필요한 경우 데이터 백업 후 TOAST 상품 가입을 진행해 주세요.(아래 상품별 이용 가능 서비스 참고) 
[자세히](https://dooray.com/home/pricing/#projectMail)

#### 개인 용량

개인 용량은 조직 사용자(멤버,손님) 개인이 각자 사용하는 저장 공간입니다.

사용자들은 자신의 개인 사용량을 (개인 설정 > 개인 사용량)에서 확인할 수 있습니다.

개인 용량 사용률이 80%, 90%,95, 100% 초과시에는 알림 메일이 발송됩니다.

#### 공용 용량

공용 용량은 조직 사용자(멤버,손님) 공동으로 사용하는 저장 공간입니다.

조직 관리자, 테넌트 관리자는 (어드민(조직 관리) > 조직 관리 > 공용 용량)에서 조직 전체의 공용 용량 사용 현황을 확인할 수 있습니다.

#### 서비스별 백업 방법

##### 사용자(멤버,손님)

###### 프로젝트(업무)
(개인/일반) 프로젝트 내 용량 산정 항목인 업무별 첨부 파일과 본문/댓글 이미지를 정리해 주세요.

*  업무 내 첨부 파일 다운로드/ 메일로 보내기 후 삭제
*  불필요한 본문/댓글 이미지 삭제

업무 본문의 경우 용량 산정 항목은 아니나, 필요한 경우 아래와 같은 방법으로 백업이 가능합니다.

1) 업무 본문과 댓글 PDF내보내기/메일로 보내기

   * 개별 업무별로 더보기 버튼(...) > PDF내보내기(메일로 보내기) 기능을 이용해 해당 업무의 본문/댓글을 백업할 수 있습니다.
   
2) 프로젝트에 등록된 업무 목록

   * 다운로드 할 프로젝트 클릭 후 화면 우상단 [다운로드] 버튼을 클릭 해 프로젝트에 등록된 (업무 제목 / 태그등) 기본 정보 다운로드가 가능합니다.
   
   * (업무 목록 다운로드시 업무 개별 본문, 댓글, 첨부 파일은 포함되지 않음)


###### 위키
위키 내 용량 산정 항목인 첨부 파일과 본문/댓글 이미지를 정리해 주세요.

* 위키 페이지 내 첨부 파일 다운로드 후 삭제 
* 불필요한 본문/댓글 이미지 삭제

###### 드라이브
* 업로드 된 파일 다운로드 후 삭제
* 휴지통 비우기 (영구 삭제)

###### 메일
개별 구성원(멤버, 손님)은 메일함을 백업/삭제해 개인 데이터를 정리해 주세요. 

* 메일 내보내기 후 삭제

1) 메일 폴더 우측 더보기(...) 클릭 > 내보내기(백업)  
2) IMAP을 통한 아웃룩 내보내기 후 삭제
   * [IMAP을 이용한 Outlook 메일 이전 가이드](https://docs.toast.com/ko/Dooray/Mail/ko/service-guide/#imap-outlook)를 참고해서 아웃룩 연동 하신 뒤 Dooray!메일 내보내기 설정을 해주시면 됩니다.

* 휴지통 비우기 (영구 삭제)

###### 캘린더
* 내 캘린더(공유 받은 캘린더는 제외)의 일정 내  첨부 파일 다운로드 후 삭제 / 불필요한 본문 이미지 삭제
* 불필요한 구독 캘린더 해지

등록된 일정의 경우 용량 산정 항목은 아니나, 필요한 경우 내보내기를 통해 백업이 가능합니다.

1) 백업할 캘린더 우측 톱니바퀴 설정 아이콘 클릭

2) 내보내기 클릭 


###### 메신저
* 나와의 대화방: 첨부 파일 삭제 (상시 삭제 가능)
* 일반 대화방: 
   * 첨부 파일 삭제 가능한 경우 다운로드 후 삭제 처리
   * (Business상품의 경우)조직 관리자가 대화방 내 첨부 파일 전송 크기/보관 기간 설정 변경 : 어드민> 테넌트 관리> 메신저 설정> 첨부 파일


###### 주소록
주소록의 경우 용량 산정 항목은 아니나, 필요한 경우 내보내기를 통해 백업이 가능합니다.
주소록 좌측 하단 **[연락처 내보내기]** 기능을 이용해 주세요.


##### 관리자

###### 프로젝트 (공용 용량)
관리자는 어드민에서 조직 내 생성된 일반 프로젝트를 모두 확인 후 불필요한 프로젝트나 데이터를 정리할 수 있습니다. (가입 상품에 따라 다름) 

1) 생성되었으나 활용하지 않는 일반 프로젝트 확인/정리: 어드민> 조직 서비스 관리 > 프로젝트 설정 > 전체 프로젝트

2) 기 '삭제'상태인 일반 프로젝트 재확인/영구 삭제 : 어드민> 조직 서비스 관리 > 프로젝트 설정 > 전체 프로젝트 > 삭제(보관) 상태 필터 후 확인
   
 
###### 메신저 (공용 용량)
(Business상품의 경우) 관리자는 어드민에서 메신저 첨부 파일의 용량과 보관 기간 설정을 변경할 수 있습니다. (어드민> 테넌트 관리> 메신저 설정> 첨부 파일)


###### 메일 대용량 첨부 (메일 공용 용량)
관리자는 어드민에서 대용량 첨부 설정을 변경하거나 (Business상품의 경우)불필요한 대용량 파일을 삭제할 수 있습니다.

1) 대용량 첨부 파일 용량/다운로드 기간 설정 변경: 어드민> 조직 서비스 관리 > 메일 설정 > 파일 첨부

2) (Business 가입의 경우)  대용량 첨부 파일 삭제: 어드민> 조직 관리 > 메일 공용 용량 > 대용량 첨부 발송
