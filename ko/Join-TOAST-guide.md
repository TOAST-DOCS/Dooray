## TOAST 상품 가입하기

무료 체험 종료 후에도 계속 Dooray!를 이용하고 싶으시면, **TOAST 상품 가입**을 진행해 주세요.  
무료체험 종료 후 7일 이내(가입 37일 이내)까지 [TOAST 상품 가입]을 하지 않으시면 이후 가입은 자동 해지됩니다.   
Dooray! TOAST 상품은 가격에 따라 아래와 같이 분류됩니다. 

![](http://static.toastoven.net/prod_dooray_project/common/common14.png)

필요한 경우 데이터 백업 후 TOAST 상품 가입을 진행해 주세요.(아래 상품별 이용 가능 서비스 참고)

### 데이터 백업 방법

#### 개인 용량

개인 용량은 구성원(멤버,손님) 개인이 각자 사용하는 저장 공간입니다.

개인 용량에 산정되는 항목은 서비스별로 아래와 같습니다.

* 프로젝트 상품 이용 시 (메일 서비스가 제외된 상품)
   * 메신저(나와의 대화방) : 첨부 파일 (이미지 포함)
   * 개인 프로젝트(업무) : 업무 내 첨부 파일, 본문/댓글 이미지
   * 개인 프로젝트 (드라이브): 업로드 된 파일 (휴지통 용량 포함)
   * 개인 프로젝트 (위키): 페이지 내 첨부 파일, 본문/댓글 이미지

* 메일 상품 이용 시 (프로젝트 서비스가 제외된 상품)
   * 메신저(나와의 대화방) : 첨부 파일 (이미지 포함)
   * 메일: 메일함 내 모든 메일 (휴지통 용량 포함)
      * 대용량 첨부 파일 발송시: 대용량 첨부 파일은 개인 용량이 아닌 메일 공용 용량에 산정되므로 개인 용량 산정시 제외)
   * 캘린더	
      * 내 캘린더: 일정 내 첨부 파일, 본문 이미지 (공유 받은 캘린더는 용량 산정 제외)
      * 구독 캘린더
   * 드라이브: 업로드 된 파일 (휴지통 용량 포함)

* 프로젝트+메일 상품 이용시 (프로젝트와 메일을 모두 사용하는 상품)
   * 메신저(나와의 대화방) : 첨부 파일 (이미지 포함)
   * 메일	메일함 내 모든 메일 (대용량 첨부 파일 제외)
   * 캘린더	
      * 내 캘린더: 일정 내 첨부 파일, 본문 이미지 (공유 받은 캘린더는 용량 산정 제외)
      * 구독 캘린더
   * 개인 프로젝트(업무) : 업무 내 첨부 파일, 본문/댓글 이미지
   * 개인 프로젝트 (드라이브): 업로드 된 파일 (휴지통 용량 포함)
   * 개인 프로젝트 (위키): 페이지 내 첨부 파일, 본문/댓글 이미지

* 메신저 lite상품 이용시
   * 메신저(나와의 대화방) : 첨부 파일 (이미지 포함)

[개인 용량 사용량 확인]

구성원들은 자신의 개인 사용량을 (개인 설정 > 개인 사용량)에서 확인할 수 있습니다.

개인 용량 사용률이 80%, 90%,95, 100% 초과시 알림 메일이 발송됩니다.

* 개인 용량: 제공되는 개인 용량입니다. 가입 상품과 계약 멤버 수에 따라 제공 용량이 달라집니다.
* 사용량: 제공되는 개인 용량 중 현재 사용중인 개인 용량입니다.
* 잔여 용량: 제공되는 개인 용량에서 사용중 용량을 차감한 용량입니다. 앞으로 사용할 수 있는 잔여 개인 용량입니다.


#### 공용 용량

공용 용량은 구성원(멤버,손님)이 공동으로 사용하는 저장 공간입니다.

공용 용량에 산정되는 항목은 서비스별로 아래와 같습니다.

* 프로젝트 상품 이용 시 (메일 서비스가 제외된 상품)
   * 메신저(일반 대화방): 첨부 파일 (이미지 포함)
   * 일반 프로젝트(업무): 업무 내 첨부 파일, 본문/댓글 이미지
   * 일반 프로젝트(드라이브): 업로드 된 파일 (휴지통 용량 포함)
   * 일반 프로젝트(위키): 페이지 내 첨부 파일, 본문/댓글 이미지
   * 일반 프로젝트(캘린더): 용량 산정 항목 아님
   * (나만의 대화방(메신저)는 개인 용량에 포함/개인 프로젝트(업무+드라이브+위키)는 개인 용량에 포함)



* 메일 상품 이용 시 (프로젝트 서비스가 제외된 상품)
   * 메신저(일반 대화방): 첨부 파일 (이미지 포함)
   * 메일(대용량 첨부 메일 쓰기 시) : 대용량 첨부 파일 (메일 공용 용량 항목으로 별도 산정)
   * (나만의 대화방(메신저)는 개인 용량에 포함 /메일 쓰기시 일반 첨부는 개인 용량에 포함)



* 프로젝트+메일 상품 이용시 (프로젝트와 메일을 모두 사용하는 상품)
   * 메신저(일반 대화방): 첨부 파일 (이미지 포함)
   * 메일	대용량 첨부 메일 쓰기 시	대용량 첨부 파일
   * 일반 프로젝트(업무): 업무 내 첨부 파일, 본문/댓글 이미지
   * 일반 프로젝트(드라이브): 업로드 된 파일 (휴지통 용량 포함)
   * 일반 프로젝트(위키): 페이지 내 첨부 파일, 본문/댓글 이미지
   * 일반 프로젝트(캘린더): 용량 산정 항목 아님
   * (나만의 대화방(메신저)는 개인 용량에 포함 /메일 쓰기시 일반 첨부는 개인 용량에 포함 /개인 프로젝트(업무+드라이브+위키)는 개인 용량에 포함)



* 메신저 lite상품 이용시
   * 메신저(일반 대화방): 첨부 파일 (이미지 포함)
   * (나만의 대화방(메신저)는 개인 용량에 포함)

공용 용량 사용률이 80%, 90%,95, 100% 초과시 관리자에게 알림 메일이 발송됩니다.

* 전체 공용 용량: 제공되는 공용 용량입니다. 가입 상품과 계약 멤버 수에 따라 제공 용량이 달라집니다.

#### 서비스별 데이터 정리 방법

##### 프로젝트(업무)
프로젝트 업무 내 용량 산정 항목인 업무 내 첨부 파일과 본문/댓글 이미지를 정리해 주세요.

- 업무 내 첨부 파일 다운로드/ 메일로 보내기 후 삭제
- 불필요한 본문/댓글 이미지 삭제

업무 본문의 경우 용량 산정 항목은 아니나, 필요한 경우 아래와 같은 방법으로 백업이 가능합니다.

1)프로젝트 업무 목록

* 프로젝트명 클릭 후 화면 우상단 다운로드 버튼을 이용해 업무의 제목 / 태그등 기본 정보 다운로드가 가능합니다.
(업무 본문 / 댓글 / 첨부 파일은 다운로드에 포함되지 않으니 참고해 주세요)

2)업무별 PDF내보내기/메일로 보내기

* 개별 업무별로 더보기 버튼(...) > PDF내보내기 or 메일로 보내기 기능을 이용해 백업이 가능합니다. 
* 내보내기 시 해당 업무의 본문 / 댓글 / 첨부 파일(메일로 보내기 시)가 모두 백업됩니다.


##### 메일
- 메일 내보내기 후 삭제
1) 메일 내보내기
메일 폴더 우측 쩜쩜쩜 버튼을 클릭 후 내보내기(백업)버튼을 누르시면 폴더별 일괄 다운로드가 가능합니다.

2) IMAP을 통한 아웃룩 내보내기
[IMAP을 이용한 Outlook 메일 이전 가이드](https://docs.toast.com/ko/Dooray/Mail/ko/service-guide/#imap-outlook)를 참고해서 아웃룩 연동 하신 뒤 Dooray!메일 내보내기 설정을 해주시면 됩니다.

##### 캘린더
